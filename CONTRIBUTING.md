# Contributing Guide

## Adding a New Script

1. Fork the repository
2. Put your script in the appropriate `scripts/category/` folder
3. Name it descriptively: `feature-name.user.js`
4. Add proper UserScript header
5. Create/update the category's README.md with description
6. Submit a Pull Request

## Pull Request Requirements

- Script must not break the game
- Include `@version`
- Test on latest game version
- Add yourself as author
- No minified/obfuscated code (unless necessary)

## Style Guide

- Use modern JS
- Add configuration object at top
- Comment key sections
