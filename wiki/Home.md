### 📦 How to Install

1. Install [Tampermonkey](https://www.tampermonkey.net/) or [Violentmonkey](https://violentmonkey.github.io/)
2. Browse to any category page from the sidebar
3. Click a script name to open it in GitHub
4. Hit **Raw** — your userscript manager will prompt you to install
